O projeto CRUD Mundo é um sistema simples para gerenciar países e cidades, com uma versão web feita em PHP, MySQL, HTML e CSS, e uma versão mobile feita em React Native integrada ao Supabase. A parte web permite cadastrar, listar, editar e excluir países, além de visualizar e gerenciar cidades pertencentes a cada país, usando rotas básicas em PHP e um banco MySQL com duas tabelas principais: “paises” e “cidades”. Já a versão mobile utiliza o Supabase como banco em nuvem, oferecendo o mesmo CRUD de países e cidades, porém com atualização em tempo real e interface construída em React Native. O objetivo do projeto é oferecer uma estrutura simples e direta para estudo de CRUD, manipulação de dados relacionais e integração entre web e mobile, mantendo uma organização básica de arquivos e funcionalidades essenciais sem complexidade desnecessária.
